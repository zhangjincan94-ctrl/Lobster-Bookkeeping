## 四、已知未跑通 / 待修复事项 ⚠️

1. **端到端联调未完成**：登录 → 创建买家 → 录入交易 → 查看列表 → 分享买家查账 全链路尚未真机/开发者工具走通。
2. **微信登录依赖**：`server/.env` 必须填入有效 `WX_APPID` 和 `WX_SECRET`，否则 `wx.login()` 拿到的 code 换不到 openid。
3. **数据库未在生产环境建表**：当前只有本地 dev 环境，生产数据库未规划。
4. **CORS / HTTPS**：开发用 HTTP，体验版/正式上线前必须 HTTPS + 配置 request 合法域名。
5. **图标资源缺失/不全**：tabBar 图标待最终美术确认，进货/供应商/统计模块图标未做。
6. **页面交互细节**：list 页面是否分页加载、下拉刷新、上拉加载均需要测试。
7. **金额小数计算**：前端用 `parseFloat` 累加，存在浮点精度风险，需要核对统计准确性。
8. **`query` 文件**：根目录存在一个内容为 "MySQL80" 的文件，疑似误提交，待确认是否删除。
9. **`AGENTS.md`**：根目录的 AGENTS.md 描述的是 C++/Qt 心电系统，与本项目无关，需要替换或删除。
10. **没有 README 部署说明**：远程仓库的 README 仅有占位内容，需要补充项目运行/部署文档。
