## 3. 端到端业务流程（必须全部跑通）`🅿️1`

- [ ] **流程 A — 商户首次使用**：新用户登录 → 填写店铺信息 → 进入工作台
- [ ] **流程 B — 完整记账**：新建买家 → 录入交易（部分付款）→ 列表查看 → 进入详情补录余款 → 状态变为已付
- [ ] **流程 C — 买家查账**：商户在买家详情点分享 → 微信发给"买家"小号 → 小号点开看到自己的所有交易和欠款
- [ ] **流程 D — 进销联动**：录入进货 → 录入销售时关联该进货 → 检查进货 available 余量减少 → 进货详情看到分摊明细
- [ ] **流程 E — 数据隔离**：商户 A 看不到商户 B 的任何数据
- [ ] **流程 F — 异常恢复**：交易提交时断网 → 提示失败 → 恢复网络后重试成功，无重复数据
